# pullrequest-first-webhook

> A GitHub App built with [Probot](https://github.com/probot/probot) that Create a pull request upon first commit to a new branch

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start
```
